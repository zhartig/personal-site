 Typescript / Javascript Versions: 8+

TS and JS are my second most used languages, and TS is my current favorite to use, and is my go to for personal projects. My main use of this language is in Angular frontends, however, I also have experience writing Node.JS applications.

For example, I have used Express to build backend services, for example, a microservice that manages an AWS Step Function workflow, including invoking, cancelling, retrying, an viewing details.

Additionally, I have built a CLI tool using TS that automatically scans a users AD group membership to discover what AWS credentials they have access to, and then automatically fetches and refreshes the credentials. This tool has become quite popular, and is not preinstalled on newly provisioned Linux developer desktops.
